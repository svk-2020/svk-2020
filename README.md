## 🌐 Мой основной проект

### 🎬 Flickollect  
Веб-приложение для управления коллекцией фильмов.

🔗 https://www.flickollect.ru  

**Что реализовано:**
- fullstack-архитектура (frontend + backend);
- сложная модель данных (фильмы, актёры, роли);
- UI: таблицы, фильтры, админ-панель;
- REST API;
- деплой и инфраструктура (Linux, Nginx, SSL).

**Стек:**
- Backend: Python, FastAPI, SQLAlchemy, PostgreSQL  
- Frontend: Next.js, React, TypeScript, Tailwind CSS  
- State: Zustand, TanStack Table  
- DevOps: Linux, Nginx, GitHub
